Cara menginstall Vim Plugin YouCompleteMe di Windows

  • Vim

Bagi kamu yang akan menginstall plugin vim youcompleteme mungkin bisa menggunakan guide ini dalam menginstall. Penulis telah merasakan pengalaman kurang enak dimana proses menginstall plugin ini kurang mulus, sering terjadi banyak error yang tidak diduga. Untuk itu langsung saja.

  1. Pastikan kamu menginstall vim dengan pasangan versi python yang benar, karena jika tidak ada kemungkinan akan muncul error seperti "youcompleteme cannot load module pythonxx.dll" dimana xx adalah versi python yang diinginkan. Biasanya penulis mendownload python dari sini. Disitu kamu bisa mengecek bagian dimana dituliskan versi python yang dibutuhkan, biasanya dituliskan dua versi python yang kamu bisa memliih salah satunya yaitu, python versi 2 atau 3.
